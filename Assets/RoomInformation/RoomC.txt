Room C Information
Install Warning Signs: Place warning signs at all slippery locations.
Anti-Slip Flooring: When repairing or installing new flooring in slippery areas, use anti-slip materials.
Install Falling Object Protection: Set up nets and facilities to prevent falling objects.
Organize for Safe Passageways: Ensure safety passages are clear and mark passages designated for workers.
Restrict Access to Unsafe Areas: Prohibit worker access to non-safety routes and install barriers as necessary.
Wear Suitable Non-Slip Safety Footwear: Workers should wear non-slip safety shoes appropriate for the floor characteristics.
Safe Ladder Use: When descending a ladder, use the same method as when climbing up.
